### 第六章 接口优化

#### 6.1 Redis预减库存减少数据库访问

- ##### 思路：减少数据库访问

  1. 系统初始化，把商品库存数量加载到redis
  2. 收到请求，redis预减库存，库存不足，直接返回，否则进入3
  3. 请求入队，立即返回排队中
  4. 请求出队，生成订单，减少库存
  5. 客户端轮询，是否秒杀成功
